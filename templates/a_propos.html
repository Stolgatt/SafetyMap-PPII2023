A propos de nous - SafetyMap

Comment utiliser le site SafetyMap ?

Le site présente sur la page principale une carte interactive sur laquelle s'affiche tous les incidents rapportés par les utilisateurs. Sur cette carte, la légende permet de sélectionner quel type d'incident vous voulez afficher, en masquant les autres. Le panneau de droite propose plusieurs choses :

  • Un bouton pour indiquer un incident dont vous auriez été témoin
  • Des statistiques sur les incidents rapportés par les utilisateurs
  • Un graphique montrant la répartition des rapports d'incidents par type d'incident

Comment signaler un incident ?

Le bouton pour signaler un incident vous renvoie à une page qui vous permet de sélectionner les informations de l'incident que vous voulez signaler. Vous pouvez ainsi décrire la date et l'heure de l'incident ainsi que son type, que vous pouvez choisir dans un menu déroulant (régulièrement mis à jour). Vous pouvez ensuite choisir sur la carte l'emplacement exact de l'incident que vous voulez signaler. Il suffit alors de double-cliquer sur la carte à l'endroit choisi pour y ajouter un marqueur.

Notez qu'il faut nécessairement être authentifié pour accéder au signalement.


Le tableau de bord

Une fois que vous vous êtes connecté, vous avez accès à votre tableau de bord qui résume vos informations personnelles, vos statistiques d'activité sur le site ainsi que les différents signalements que vous avez fait.

ATTENTION : Vous ne pouvez pas supprimer un incident que vous avez signalé. Pour garder le site précis et juste, s'il-vous-plaît, soyez sûr des informations que vous entrez.

Notre motivation

En France, 20% des Français déclarent ressentir une certaine insécurité, dont 27% parmi les femmes. De plus en plus de personnes n'osent plus sortir seules le soir et les solutions à ce problème restent limitées voire inexistantes.

En faisant ce constat auprès de nos proches et de nos camarades, nous avons décidé d'agir pour améliorer cette situation.


La question que nous avons alors décidé de traiter est la suivante :

Comment pourrions-nous faciliter l’accès à l’information sur l’insécurité dans les villes pour le bien-être de la population?

Notre solution

Cette carte interactive a pour but d'avertir les utilisateurs de SafetyMap des potentiels dangers récents observés en ville, et ce de manière collaborative et simultanée. L'idée est donc qu'à travers cet outil, chaque citoyen pourra être informé des dangers qu'il sera susceptible de rencontrer et pourra lui aussi informer les autres de ce dont il est témoin. Les utilisateurs pourront alors choisir un autre itinéraire pour éviter les mauvaises rencontres ou les retards.
